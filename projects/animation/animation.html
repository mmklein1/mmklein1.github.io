Basic :hovers and transitions

1. In order to have a :hover transition applied, you need to write two separate lines of code inside your CSS.

2. The first is to apply a :hover on the element or class

3. For example, when you hover over this paragraph the background-color changes. Notice that this change here is instant and there's no delay.

4. To have a change in the duration of how it appears, you need to apply a transition property inside the css of the class you want to affect. This is written as
transition: [transition-property] [transition-duration] [transition-timing-function] [transition-delay];
In this example since there are multiple items changing, all is used for the transition property

5. In order to have the position change, you would change the x or y position using transform:translateX(____)

6. You can also use opacity to have elements fade in or out slowly using opacity
